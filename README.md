ofxPDSPPatches
=====================================

This is a set of high level classes that builds upon [ofxPDSP](https://github.com/npisanti/ofxPDSP).  [ofxWatchFile](https://github.com/nariakiiwatani/ofxWatchFile.git) is also required for some classes to work, but not on Android and iOS.
